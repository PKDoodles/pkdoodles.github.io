Project Overview

My goal in this project was to build a working framework of a stealth game from the ground up and showcase it in a level! I am really happy with how it turned out! I placed focus on making a more complex AI system as well as a well design UI to help the player understand the mechanics! Although this project was a rough protoype, it is my favourite project yet as it taught me a lot about aspects of Ureal Engine 5 I did not know and gave me a basis to create my own games without the need of a framework!

This project gave me a very strong technical improvement as I now understood UI, AI, Input Actions and Animations which are very strong abilities which I am willing to pursue a lot in the future!
